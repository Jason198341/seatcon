도움말

이 페이지는 개발 또는 테스트 환경에서 API 키를 쉽게 설정하기 위한 도구입니다.

보안을 위해 프로덕션 환경에서는 다음 방법 중 하나를 사용하는 것이 좋습니다:

  1. 프로젝트 루트에 .env 파일 생성 (권장)
  2. 서버 실행 시 환경 변수 설정
  3. 클라우드 서비스 설정에서 환경 변수 관리

프로덕션 환경에서는 이 페이지를 제거하거나 접근을 제한하세요.
